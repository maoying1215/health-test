# 心理-代谢健康管理小程序

## 项目简介
这是一个用于职业人群心理-代谢共病管理的微信小程序,提供健康评估、数据追踪、社区互动等功能。

## 功能模块

### 1. 首页 (pages/home)
- 快速评估入口(心理评估、代谢评估、综合评估)
- 健康概览展示
- 今日健康建议
- 功能快捷入口

### 2. 健康评估 (pages/assessment)
- 心理健康评估问卷
- 代谢健康评估问卷
- 综合健康评估
- 评估结果展示与分析
- 个性化健康建议

### 3. 健康数据 (pages/health-data)
- 健康数据录入(体重、身高、血糖、血压)
- BMI自动计算
- 历史记录查看
- 数据趋势分析

### 4. 健康社区 (pages/community)
- 健康经验分享
- 分类浏览(心理健康、代谢健康、生活方式)
- 互动功能(点赞、评论)

### 5. 个人中心 (pages/profile)
- 用户信息管理
- 健康统计数据
- 功能设置
- 账号管理

## 技术特点

1. **无AI依赖**: 所有健康建议和评估结果基于预设规则生成,不依赖外部AI服务
2. **本地存储**: 使用微信小程序本地存储功能保存用户数据
3. **纯前端实现**: 不需要后端服务器,可直接在微信开发者工具中运行
4. **响应式设计**: 适配不同尺寸的手机屏幕

## 使用说明

### 1. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目目录,填写AppID(测试可使用测试号)
4. 点击"导入"

### 2. 运行项目
1. 导入成功后,项目会自动编译
2. 在开发者工具中预览小程序效果
3. 可以点击"预览"使用真机测试

### 3. 注意事项
- 图标图片需要自行准备,放在 `images` 目录下
- tabBar需要的图标文件:
  - home.png / home-active.png
  - assessment.png / assessment-active.png
  - data.png / data-active.png
  - community.png / community-active.png
  - profile.png / profile-active.png
- 建议尺寸: 81px * 81px

### 4. 自定义配置
可以在以下文件中进行自定义:
- `app.json`: 修改页面配置、导航栏样式等
- `app.wxss`: 修改全局样式
- `utils/util.js`: 修改评估算法和建议生成逻辑

## 数据存储说明

项目使用微信小程序的本地存储功能,存储以下数据:
- `userInfo`: 用户基本信息
- `assessmentHistory`: 评估历史记录
- `healthData`: 健康数据记录

数据仅保存在用户设备本地,不会上传到服务器。
